<objective>
Build the data validation gate as four LOGICAL gates across three prebuild scripts (scripts/validate-data.mjs runs Ajv structural validation + referential integrity + the imperative date-range order check; scripts/check-editorial.mjs and scripts/check-images.mjs are plan 01-06). Every failure path ships with a negative fixture that proves the build fails on bad input. A gate is only real if its failure path is tested AND it proves it scanned a non-zero corpus.
